1. [expression for item in iterable] => 

Equivalent to :

result = []
for item in iterable:
    result.append(expression)

Example 1: square of numbers

nums = [1,2,3,4]
squares = [n*n for n in nums]

Equivalent to : 

nums = [1,2,3,4]
squares = []
for n in nums:
    squares.append(n*n)



2.Example 2: With Condition 
[expression for item in iterable if condition]

nums = [1,2,3,4,5,6]
evens = [n for n in nums if n % 2 == 0]

equivalent to :

nums = [1,2,3,4,5,6]
evens = []
for n in nums:
    if n % 2 == 0:
        evens.append(n)

3.Example 3: If-Else inside comprehension 
[expr_if_true if condition else expr_if_false for item in iterable]

nums = [1,2,3,4]
labels = ["even" if n % 2 == 0 else "odd" for n in nums]

4. Nested List Comprehension

matrix = [[1,2],[3,4],[5,6]]
flat = [num for row in matrix for num in row]

Equivalent to :

flat = []
for row in matrix:
    for num in row:
        flat.append(num)


5. Lambda functions

normal function

def square(x):
    return x*x

    equivalent to:

square = lambda x: x*x
lambda arguments: expression(syntax)


| Situation            | Use                |
| -------------------- | ------------------ |
| Transform list       | List comprehension |
| Add condition        | List comprehension |
| Sorting custom rule  | Lambda             |
| Quick small function | Lambda             |
| Complex logic        | Normal function    |

sorting :

pairs = [(1,3),(2,1),(4,2)]
pairs.sort(key=lambda x: x[1])

nums.sort(key=lambda x: (-x[1], x[0]))

Example:

Sort by frequency descending
Then by value ascending
Lambda is very common in sorting problems.
